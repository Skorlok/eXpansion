/admin player kick kick kick the player from the server
/admin player kick #login will kick the player from the server. A kicked player may return to the server whanever he desires.
/admin player ban ban Ban the player from the server
/admin player ban #login will ban the player from the server. He may not return until the server is restarted
/admin player black black Add the player to the black list
/admin player black #login will add the player to the blacklist of this server. He may not return until the server blacklist file is deleted. Other server might use the same blacklist file!!
/admin player remove ban unban Removes the ban of the player
/admin player remove ban #login will remove the ban of the player from this server He may rejoin the server after this.
/admin clear banlist cleanbanlist clears the banlist of players
Will completeley clear the banlist. All banned players will be able to rejoin the server.
/admin get banlist getbanlist shows the current banlist of players
/admin clear blacklist cleanblacklist clears the blacklist of players
Will completeley clear the blackList. All blacklist players will be able to rejoin the server.
/admin get blacklist getblacklist shows the current banlist of players
/admin get guestlist getguestlist shows the current guest of players
/admin get ignorelist getignorelist shows the current ignorelist of players
/admin remove black unblack Removes the player from the black list
/admin player remove black #login will remove the player from the servers blacklist He may rejoin the server after this.
/admin player spec spec Forces the player to become spectator
/admin player spec #login The playing player will be forced to become a spectator If the max spectators is reached it the player won't become a spectator
/admin player ignore ignore Adds player to ignore list and mutes him from the chat
/admin player ignore #login will ignore the players chat This player won't be able to communicate with other players.
/admin player unignore unignore Removes player to ignore list and allows him to chat
/admin player unignore #login will allow this player to use the chat again This player will be able to communicate with other players
